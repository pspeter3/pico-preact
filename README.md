# pico-preact

PicoCSS Preact Template

## Use this template

This repository is a GitHub template intended to be created via GitHub's **Use this template** button.

1. Open this repository on GitHub.
2. Click **Use this template**.
3. Choose **Create a new repository**.
4. Enter your new repository name, visibility, and owner.
5. Click **Create repository from template**.
6. Clone your new repository and start customizing.
